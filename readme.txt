== Description ==

This plugin will help you to set the buy x and get y product offer rules in your WooCommerce site.

== DISCOUNT TYPES AND FEATURES ==

* Site admin can set rule buy one get one product for the customer. It will helps to improve products sale. You can set the product as free or percentage discount or fixed price discount based on your need. The discounted product will be removed when the parent product is removed from the cart. The quantity of the discounted product will update based on the parent product quantity and the discounted product quantity button will disabled in cart.

= Percentage Discount Example =
* Buy laptop from Category Asus and save 10% for Asus laptop bag recursively

= Free Discount Example =
* Buy one Cap And Get one Cap free recursively

= Fixed Discount Example =
* Buy one Belt And Get one Belt with 20 OFF recursively

== Installation ==

Download the plugin in your directory. Upload it in your site add plugin page and active the plugin. Also you can manually copy the plugin folder in your site plugin folder PATH(your_wordpress_path/wp-content/plugins) and you can active the plugin.  

= Minimum Requirements =

* WordPress 4.0.0 or greater
* WooCommerce 3.0.0 or greater
* PHP version 5.6.0 or greater
* MySQL version 5.0 or greater
